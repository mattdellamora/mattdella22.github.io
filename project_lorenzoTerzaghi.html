Lorenzo Terzaghi's website
Website development, 2021

Lorenzo is a graphic designer and photographer based in Ticino. As a working professional, he needed a website to display his work and wanted a solution where he did not have to care much about the technical aspects of mantaining a site. So I developed this website which displays the projects that Lorenzo feeds into a user-friendly backend, Wordpress. The portfolio updates then by itself based on the content defined within Wordpress. Clic here to visit his website. Credits to Lorenzo, for the cover image of this project.
