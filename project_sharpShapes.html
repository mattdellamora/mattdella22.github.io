Sharp Shapes
SUPSI, 2019

This is the output of a workshop held by Parco Studio (Milan). An interactive webpage, where with the simple interaction of a clic, the user decides which one out of three elementary shapes to use for a composition inside a grid. I had the opportunity to work with and develop some basic pre-fabricated code snippets, in order to produce something that is visually appealing and is the beginning of more complex things.

Web design, Javascript, Interaction
